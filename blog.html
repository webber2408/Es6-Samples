ECMAScript is the scripting standard by ECMA International of which JavaScript is the best-known implementation. Given below is a detailed explanation along with examples of how ECMAScript escalated from 2015 to 2019 and what all important features were released in these years. ECMAScript 5 | ES5 1) Arrow Functions Arrow functions are the alternatives to normal functions in ECMAScript which doesn't have its own bindings to this or super. Example: [sourcecode language="javascript"] var greet = (message) => { console.log("Hello, " + message); } greet("it's nice meeting you!"); [/sourcecode] Output: 
Hello, it's nice meeting you!
2) Classes A class is a special type of function which can contain multiple properties and functions within. Example: [sourcecode language="javascript"] class Employee{ constructor(empName){ this.empName = empName; } getEmployeeName(){ console.log("Employee Name: " + this.empName); } } const emp = new Employee("John"); emp.getEmployeeName(); [/sourcecode] Output: 
Employee Name: John
